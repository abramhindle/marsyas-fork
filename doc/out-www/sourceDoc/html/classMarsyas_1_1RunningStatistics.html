Detailed Description

Gathers the running average, variance, standard deviation, etc.

Outputs the running average and standard deviation of all the input so far.

Controls:

  • mrs_bool/enableMean: enable outputting of the mean values (on by default).
  • mrs_bool/enableStddev: enable outputting of the standard deviation values (on by default).
  • mrs_bool/enableSkewness: enable outputting of the skewness values (off by default).
  • mrs_bool/clear: clear the internal buffers to start fresh.
  • mrs_book/clearPerTick: clear the buffers at the start of each tick. This disables the "running" property of the calculations.
