Spectral Flatness Measure.

Spectral Flatness Measure is a feature set defined in the MPEG-7 audio standard. It assumes an 1/4 octave frequency resolution, resulting in 24 frequency bands between 250Hz and 16kHz. Based on code provided by Luis Gustavo Martins lmart.nosp@m.ins@.nosp@m.inesc.nosp@m.port.nosp@m.o.pt
