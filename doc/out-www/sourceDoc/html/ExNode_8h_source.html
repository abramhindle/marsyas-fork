19 /***
20 ExNode represents an expression tree node. Additional nodes may be added later,
21 but may also need to be added to the parser.
22 
23 There should only ever exist a single parent of any node, that is, a node may
24 only be referenced by one object.
25 
26 To add library functions add a line like this to ExParser.h::preload() :-
27 library->addRecord("Real|R.cos(mrs_real)|cos(mrs_natural)",new ExRecord(T_FUN,new ExNode_RealCos("mrs_real","Real.cos(mrs_real)"),true));
28 then define the function as an ExNode class, you must support the calc and copy
29 functions. In the constructor make sure you set is_pure to true if the function
30 can be reduced to a const value given const parameters, or false otherwise.
31

35  virtual ExVal calc() { return false; }
36  ExNode* copy() { return new ExNode_NatDbl(type,signature); }
37 };
