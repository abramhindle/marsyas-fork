Detailed Description

Delay.

Simple Delay example; in the first block, it will adapt the delay from 0 to the actual delay

Controls:

  • mrs_real/maxDelaySamples [w] : maximum delay in samples.
  • mrs_real/maxDelaySeconds [w] : maximum delay in seconds.
  • mrs_real/delaySamples [w] : delay in samples.
  • mrs_realvec/delaySamples [w] : multiple delays in samples.
  • mrs_real/delaySeconds [w] : delay in seconds.
  • mrs_realvec/delaySeconds [w] : multiple delays in seconds.
